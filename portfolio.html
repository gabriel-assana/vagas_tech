Sobre o projeto

O Vagas Tech foi concebido para a conclusão do curso de Front-end Coding da Estação Hack from Facebook em parceria com a Digital House Coding School. O objetivo de equipe de desenvolvedores foi criar uma plataforma para programadores Front-End e Back-End encontrarem vagas de trabalho, já que atualmente, há uma carência de sites focados nesse publico tão vasto. Visamos trazer a máxima funcionalidade e praticidade somada com uma estética agradável. O diferencial do nosso projeto é a disponibilidade dos usuários criarem uma conta para guardar suas vagas desejadas e acompanhá-las. Esperamos que todos consigam ter uma boa experencia com a Vagas Tech!
